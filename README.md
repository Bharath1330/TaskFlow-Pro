# 🚀 TaskFlow Pro - Smart Task Manager

A modern, responsive, full-stack Task Management Web Application built using **Python Flask**, **SQLite**, **HTML**, **CSS**, **JavaScript**, and **Bootstrap**.

TaskFlow Pro helps users organize their daily work with categories, priorities, deadlines, progress tracking, and a clean responsive interface.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Bharath1330/TaskFlow-Pro.git
```

Move into the project directory

```bash
cd TaskFlow-Pro
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
